Умова задачі

Умова лабораторної роботи №1 передбачає виконання п’яти завдань мовою C# в консольному застосунку згідно з варіантом. Виконання завдання 1 вимагає використання методів класу Console. Виконання завдання 2 вимагає використання методів класу Math. Виконання завдання 3 вимагає використання оператору if Виконання завдання 4 вимагає використання оператору switch Виконання завдання 5 вимагає використання оператору циклу

Завдання з лабораторної

Аналіз задачі

Для виконання завдання необхідно створити switch case для того, щоб обирати одне завдання з 5, яке буде виконуватися. Для першого завдання потрібно проаналізувати задачу, вирішити її та створити формулу для її виконання. Для другого завдання для вирішення прикладу будемо використовувати математичні функції класу Math. Для третього завдання нам знадобиться оператор if-else. Для четвертого завдання необхідно створити switch case для того, щоб виводити один із пальців руки. Для п'ятого завдання будемо використовувати цикл for для того, щоб порахувати суму.

            Console.WriteLine("Введіть b: ");

            }
        }

        static void task5()
        {
            Console.WriteLine("===Task 5===");
            Console.WriteLine("Введiть n: ");
            int n = Convert.ToInt32(Console.ReadLine());
            Console.WriteLine("Введiть x: ");
            int x = Convert.ToInt32(Console.ReadLine());
            if (x <= 0 || n <= 0 || n < x) Console.WriteLine("Невірні введені числа, спробуй ще раз!\n");
            else
            {
                double sum = 0;
                for (int i = 1; i <= n; i++)
                {
                    double numerator = Math.Pow(-1, i) * Math.Sqrt(Math.Pow((x + 1), i));

                    double denominator = Math.Pow(i, x) - Math.Pow(x, i);

                    double member = numerator / denominator;
                    sum += member;
                }
                Console.WriteLine("\nСума = " + sum);
            }
